# COVID-19 Analytics Dashboard

An interactive COVID-19 Analytics Dashboard built using Python, Streamlit, Pandas, and Plotly.

## Features

- Country-wise COVID-19 statistics
- Date range filter
- Total Cases, Deaths, and Population KPIs
- Daily New Cases visualization
- Daily Deaths visualization
- Vaccination trend analysis
- Interactive dashboard built with Streamlit

## Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
- Jupyter Notebook

## Dataset

The dataset is not included because it exceeds GitHub's file size limit.

Download it from:

https://covid.ourworldindata.org/data/owid-covid-data.csv

Place it in:

```
data/owid-covid-data.csv
```

## Run the Project

```bash
pip install -r requirements.txt
streamlit run dashboard/dashboard.py
```
